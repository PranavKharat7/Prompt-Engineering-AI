üîπ Definition and Core Concepts

Menu Actions Pattern

Structures AI interactions like a menu-based system. Instead of free-form responses, the AI presents a list of clear, labeled options for users to choose from. This pattern transforms an open-ended conversation into an interactive, guided experience.

üå≥

Branching Conversations

Each menu option leads to a different path or type of response, much like navigating a website or app with clear navigation trees.

üéÆ

User-Driven Navigation

Users stay in control by picking what they want next. This reduces cognitive load and makes AI interactions feel organized and intuitive.
